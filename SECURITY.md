# Security Policy

## Supported Versions
v26.5.x

## Reporting a Vulnerability
Contact maintainers directly. Do NOT open public issues.

## Responsible Disclosure
90 days

## Receipt-Gated Security Changes
Is receipt-gating required? **true**

## Out-of-Membrane Mutation Policy
All Out-Of-Membrane mutations are blocked and require autonomic remediation or manual exception receipts.

## Verification Commands
```bash
pnpm run ghf:verify --strict
```
